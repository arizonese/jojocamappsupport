# JoJo Cam - App Support Website

A modern, static HTML website for the JoJo Cam P2P camera application.

## Pages

- **Home** (`index.html`) - Landing page showcasing JoJo Cam features
- **About** (`about.html`) - Information about the app and its technology
- **Support** (`support.html`) - Support page with embedded feedback form
- **Privacy Policy** (`privacy.html`) - Full privacy policy

## Getting Started

Simply open `index.html` in your web browser or deploy the files to any web hosting service.

### Local Development

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the menu

### Deployment

These are static HTML files with no dependencies. You can host them on:
- GitHub Pages
- Netlify
- Vercel
- Any static web hosting service

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern, clean UI with gradient accents
- Embedded Google Form for support feedback
- Privacy-focused design
- Fast loading (no external dependencies required)

## Files

```
jojocamappsupport/
├── index.html      # Landing page
├── about.html      # About page
├── support.html    # Support page with embedded form
├── privacy.html    # Privacy policy
├── styles.css      # Main stylesheet
└── README.md       # This file
```

## Customization

You can easily customize:

- **Colors** - Edit the CSS variables in `styles.css` (`:root` section)
- **Content** - Edit the HTML files directly
- **Form** - Replace the Google Form URL in `support.html` with your own form
